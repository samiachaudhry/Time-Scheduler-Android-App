# Time-Scheduler-Android-App
Android application for managing daily schedules and tasks.

# Overview
The Time Scheduler is an Android application developed to help users efficiently organize and manage their daily schedules and tasks. The application provides an intuitive interface for creating, updating, and managing personal activities, allowing users to improve productivity and time management.
This project was developed as part of an academic software engineering project to strengthen Android application development skills and apply software design principles in a real-world scenario. 

# Features
- Create and manage daily schedules
- Add, edit, and delete tasks
- Organize activities efficiently
- User-friendly Android interface
- Persistent data management
- Simple and responsive application design

# Technologies Used
- Programming Language: Java
- IDE: Android Studio
- Database: MySQL
- User Interface: XML
- Build System: Gradle
